# Team Task Manager

A full-stack Team Task Manager application built using:

- React
- Node.js
- Express.js
- MongoDB
- Railway
- Vercel

## Features

- User Signup/Login
- Dashboard
- Create Projects
- Create Tasks
- Assign Members
- Task Status
- Delete Tasks
- MongoDB Database
- Full Stack Authentication

## Tech Stack

Frontend:
- React
- Axios
- React Router DOM

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation

### Frontend

cd client
npm install
npm run dev

### Backend

cd server
npm install
npm run dev
